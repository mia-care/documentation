_March 18th, 2026_

The minor release v2.5 comprehends 
- **Test Management**: Streamlined workflows for test suite composition and full support for manual and automatic tests with enhanced differentiation between manual and automatic testing management. 
- **AI-powered quality evaluation** with automatic deprecation of outdated evaluations at relevant information modification. 
- **Security & Compliance**: Advanced vulnerability visualization with filtered views for implemented software items.
- **User Experience**: Improved sorting and navigation of Software System Versions.
